 # Maschinenlesbare chemische Strukturen

import useBaseUrl from '@docusaurus/useBaseUrl';


## Einleitung

Die Suche nach relevanten Artikeln auf der Grundlage von IUPAC-Namen oder Trivialnamen von Molekülen ist eine schwierige Aufgabe, während chemische Identifikatoren eine eindeutige Identifizierung von Verbindungen ermöglichen. Das Neuzeichnen chemischer Strukturen ist arbeits- und zeitintensiv, während Dateien mit chemischen Tabellen (CT) ohne zusätzlichen Aufwand mit jeder gängigen Strukturzeichnungssoftware verwendet werden können. 

Die Bereitstellung maschinenlesbarer chemischer Strukturen als [CT-Dateien](/docs/ct_files) wie mol-Dateien, [InChI](/docs/inchi)-Kennungen und [SMILES](/docs/smiles)-Strukturcodes als Teil eines mit einem Forschungsartikel verbundenen Datensatzes verbessert dessen [Auffindbarkeit](/docs/fair), indem der Artikel leicht indizierbar und nach Strukturen durchsuchbar wird. Dies verbessert auch die [Interoperabilität](/docs/fair) und erleichtert die [Wiederverwendung](/docs/fair) der wissenschaftlichen Arbeit.

Im Folgenden wird eine Anleitung gegeben, wie man die Maschinenlesbarkeit von Chemie-Forschungsartikeln erhöhen kann, indem maschinenlesbare chemische Strukturen als Datendateien innerhalb des zugehörigen Datensatzes bereitgestellt werden. Darüber hinaus werden Empfehlungen zur Bereitstellung von Strukturcodes und Identifikatoren in einer maschinenlesbaren Zusatztabelle innerhalb des zugehörigen Datensatzes gegeben. 

Während solche Informationen für strukturierte domänenspezifische Repositorien wie Chemotion Reopositoy nicht erforderlich sind, da diese Informationen von der Repository-Software bereitgestellt werden, profitieren Datensätze in allgemeinen Repositorien von diesen Informationen. 

Bitte beachten Sie, dass es derzeit Einschränkungen bei [SMILES](/docs/smiles) und [InChI](/docs/inchi) für anorganische und insbesondere metallorganische Verbindungen gibt. 

## Mol-Dateien abrufen

Alle gängigen Strukturerstellungsprogramme speichern mol-Dateien. Kopieren Sie die Struktur in ein neues Dokument in Ihrer bevorzugten Strukturierungssoftware. Wählen Sie dann Datei -> Speichern unter -> wählen Sie MDL Molfile aus dem Dropdown-Menü -> Speichern.

Der Name der Datei kann in Anlehnung an Ihren Laborjournaleintrag und die Nummerierung der Struktur im Artikel gewählt werden.

## SMILES, InChI und InChIKey abrufen

### ChemDoodle

Um SMILES, InChI und InChIKey in ChemDoodle abzurufen, markieren Sie eine Struktur und wählen dann Bearbeiten -> Kopieren als -> Daylight SMILES oder IUPAC InChI. 

![ChemDoodle_SMILES_InChI_InChIKey](/img/topics/Screenshots_ChemDoodle_SMILES_InChI_InChIKey.png)

(ChemDoodle v11.7.0, iChemLabs, LCC., Chesterfield, VA, Vereinigte Staaten, 2021.)

Sie können auch eine Struktur auswählen und dann Struktur -> Zeilennotation generieren -> Daylight SMILES oder IUPAC InChI wählen.

Alternativ können Sie SMILES, InChI und InChIKey auch als Textdateien speichern, indem Sie Datei -> Speichern unter -> Daylight SMILES oder InChI aus dem Dropdown-Menü wählen -> Speichern.

Wenn Sie "IUPAC InChI" wählen, wird auch der InChIKey angezeigt, wenn er in den Voreinstellungen aktiviert ist. Um InChIKey einzuschließen, wählen Sie Bearbeiten -> Voreinstellungen -> Registerkarte Dateien -> scrollen Sie nach unten und markieren Sie InChI-Schlüssel einschließen. 

### ChemDraw Professional

Um SMILES, InChI und InChIKey in ChemDraw Professional abzurufen, markieren Sie eine Struktur und wählen dann Bearbeiten -> Kopieren als -> SMILES, InChI oder InChIKey.

![ChemDoodle_SMILES_InChI_InChIKey](/img/topics/Screenshots_ChemDraw_SMILES_InChI_InChIKey.png)"

(ChemDraw Professional v20.1.0.11, PerkinElmer Informatics, Inc., Waltham, MA, Vereinigte Staaten, 2021.)

### ChemSketch

Um SMILES, InChI und InChIKey in ACD/ChemSketch abzurufen, markieren Sie eine Struktur und wählen Sie dann Werkzeuge -> Generieren -> SMILES-Notation oder InChI für Struktur.

![ChemDoodle_SMILES_InChI_InChIKey](/img/topics/Screenshots_ChemSketch_SMILES_InChI_InChIKey.png)

(ACD/ChemSketch v2021.1.1, Advanced Chemistry Development, Inc., Toronto, ON, Kanada, 2021). 

Wenn Sie InChI für Structure auswählen, wird auch der InChIKey angezeigt, wenn er unter InChI Options aktiviert ist. Um den InChIKey einzubeziehen, wählen Sie Extras -> Generieren -> InChI-Optionen und markieren Sie InChI-Schlüssel. 

### MarvinSketch

Um SMILES, InChI und InChIKey in MarvinSketch abzurufen, markieren Sie eine Struktur und wählen dann Bearbeiten -> Kopieren als. In einem neuen Fenster wählen Sie Daylight SMILES, InChI/RInChI oder InChIKey/RInChIKey.

![ChemDoodle_SMILES_InChI_InChIKey](/img/topics/Screenshots_MarvinSketch_SMILES_InChI_InChIKey.png)

(MarvinSketch v21.18, ChemAxon, Ltd., Budapest, Ungarn, 2021.)

Alternativ können SMILES, InChI und InChIKey auch als Textdateien gespeichert werden, indem Sie Datei -> Speichern unter -> Daylight SMILES, InChI/RInChI oder InChIKey/RInChIKey aus dem Dropdown-Menü wählen -> Speichern.

## Maschinenlesbare Daten als ergänzende Tabelle bereitstellen

Während die Datendateien direkt zum Datensatz hinzugefügt werden können, können zusätzliche Informationen zur Verbesserung der Maschinenlesbarkeit als [supplementary table] (https://doi.org/10.1186/s13321-021-00520-4) bereitgestellt werden. Diese Tabelle sollte auch in einem maschinenlesbaren, textbasierten Format wie z. B. einer .csv-Datei bereitgestellt werden.

Es wird empfohlen, diese Tabelle um die folgenden Spalten zu ergänzen:
- lokaler chemischer Strukturidentifikator, d. h. die Nummern der Strukturen im Forschungsartikel
- lokales chemisches Probensymbol, d.h. Buchstabencode und Nummer in Ihrem Laborjournal
- SMILES
- InChI und InChIKey

Der Buchstabencode und die Nummer in Ihrem Laborjournal sollten enthalten sein, da analytische Datendateien in einem Datensatz häufig nach den Einträgen im Laborjournal benannt werden.

Zusätzlich kann diese Tabelle noch weitere Spalten enthalten für:
- IUPAC-Name
- Synonyme, d. h. gebräuchliche Namen einer Verbindung
- PubChem-Verbindungsidentifikator
- CAS-Registernummer
- Kommentar (falls erforderlich)

PubChem-Verbindungsbezeichner (CIDs) sowie CAS-Registernummern lassen sich in [PubChem](https://pubchem.ncbi.nlm.nih.gov/) leicht durch eine Suche über SMILES, InChI oder InChIKey abrufen. 

Vorlagen für eine solche Tabelle werden als [.ods](Pfadname://../../../assets/Supplementary_table_template.ods), [.xlsx](Pfadname://../../../assets/Supplementary_table_template.xlsx) und [.csv](Pfadname://../../../assets/Supplementary_table_template.csv) Dateien bereitgestellt. Diese Vorlagendateien nutzen auch die Vorteile von [ontologies](/docs/ontology), um Begriffe für Menschen und Maschinen eindeutig zu identifizieren. 

Wenn die experimentelle Arbeit in einem [ELN](/docs/eln) dokumentiert ist, können diese Informationen auch vom ELN-System bereitgestellt werden. Chemotion ELN generiert [SMILES](/docs/smiles), [InChI](/docs/inchi), InChIKey sowie RInChI und RInChIKey für Verbindungen und Reaktionen. Diese Informationen sind auch im Chemotion Repository verfügbar, d.h. eine solche Zusatztabelle ist bei strukturierten domänenspezifischen Repositories wie Chemotion Repository nicht erforderlich, während Datensätze in generischen Repositories von einer solchen Tabelle profitieren.
